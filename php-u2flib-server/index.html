php-u2flib-server

Serverside U2F library for PHP. Provides functionality for registering tokens and authentication with said tokens.

Dependencies

Dependencies are managed with composer, see https://getcomposer.org/ for information about obtaining that. To setup dependencies when composer is availabel, run:

$ composer.phar install

Tests

To run the testsuite phpunit is required, to run it:

$ phpunit
